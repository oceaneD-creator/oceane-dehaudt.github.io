XSS LAB

Projet de laboratoire XSS sécurisé.

Popup XSS

Injection JavaScript via le champ de recherche de OWASP Juice Shop. Test réalisé dans un environnement local sécurisé afin d'observer l’exécution de code côté client.

Page compromise

Démonstration d’une modification du DOM via une charge XSS. Simulation d’un scénario de compromission visuelle d’interface.

Docker

Déploiement du laboratoire OWASP Juice Shop via Docker afin d’obtenir un environnement de test isolé.

Mitigation

Analyse des méthodes de protection contre les attaques XSS : validation des entrées, encodage HTML et Content Security Policy.
